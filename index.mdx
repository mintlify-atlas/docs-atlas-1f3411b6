nvim-luffy is a meticulously crafted Neovim configuration that transforms your editor into a powerful development environment. Like the legendary pirate seeking the One Piece, this configuration helps you navigate the vast seas of code with speed, precision, and style.

## What is nvim-luffy?

nvim-luffy is a complete Neovim configuration built with modern Lua practices and powered by [lazy.nvim](https://github.com/folke/lazy.nvim) for blazing-fast plugin management. It provides a carefully curated set of plugins and keybindings designed for productive development across multiple languages.

## Key Features

### Lightning-Fast Startup
Optimized plugin loading with lazy.nvim ensures your editor is ready when you are. Most plugins load on-demand, keeping startup times minimal.

### Full-Featured LSP
Out-of-the-box support for:
- TypeScript/JavaScript (ts_ls)
- HTML, CSS, Tailwind
- Lua (lua_ls)
- Python (pyright)
- GraphQL, Svelte, Prisma
- And more via Mason

### Intelligent Completion
Powered by nvim-cmp with:
- LSP-based completions
- Snippet support via LuaSnip
- Path and buffer completions
- VS Code-like pictograms

### Smart Code Navigation
- **Telescope**: Fuzzy find files, grep through projects, search recent files
- **Harpoon**: Mark and instantly jump to your most important files
- **Treesitter**: Advanced syntax highlighting and code understanding
- **Flash**: Jump to any location with minimal keystrokes

### Developer Experience
- **Formatting**: Automatic code formatting with Prettier, Stylua, Black
- **File Explorer**: nvim-tree with intuitive keybindings
- **Autopairs**: Intelligent bracket and quote pairing
- **Trouble**: Beautiful diagnostics and quickfix lists
- **Alpha**: Custom dashboard with One Piece-themed ASCII art

## Philosophy

nvim-luffy follows these core principles:

1. **Minimal but Complete**: Only include plugins that add real value
2. **Speed First**: Lazy-load everything possible
3. **Sane Defaults**: Works great out of the box, customizable when needed
4. **Modern Lua**: Built entirely in Lua with clean, maintainable code
5. **Keyboard-Driven**: Efficient keybindings with Space as leader key

## Project Structure

```lua
~/.config/nvim/
├── init.lua                    # Entry point
└── lua/
    └── leyland/
        ├── core/
        │   ├── init.lua        # Loads core modules
        │   ├── options.lua     # Vim options
        │   ├── keymaps.lua     # Global keybindings
        │   └── autocommands.lua # Autocommands
        ├── lazy.lua            # Lazy.nvim bootstrap
        ├── lsp.lua             # LSP configuration
        └── plugins/            # Plugin specifications
            ├── init.lua
            ├── alpha.lua       # Dashboard
            ├── telescope.lua   # Fuzzy finder
            ├── nvim-cmp.lua    # Completion
            ├── harpoon.lua     # File navigation
            ├── treesitter.lua  # Syntax highlighting
            └── lsp/
                ├── mason.lua   # LSP installer
                └── lsp.lua     # LSP setup
```
